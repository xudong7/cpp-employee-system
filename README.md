# 中山大学软件工程学院初级实训第二次项目
## 公司人员和薪酬管理系统

## 操作说明：
### 进入系统，读取employees.txt中的员工数据
![alt text](<屏幕截图 2024-10-30 214746.png>)
### 输入0 - 5选择操作
#### 1 => 输入所有员工数据，同时抹除原有员工数据（需要事先确认）
##### 输入员工名字即可，若无新的员工，则输入“@”即可
![alt text](<屏幕截图 2024-10-30 214842.png>)
#### 2 => 增加新员工数据
##### 以“name[space]grade”形式输入，如“Tom 3”表示名为Tom的Technician（grade为3）
![alt text](<屏幕截图 2024-10-30 214914.png>)
#### 3 => 删除员工数据（根据ID删除）
##### 输入要删除员工的ID
![alt text](<屏幕截图 2024-10-30 214934.png>)
#### 4 => 输入工作时间和销售额
##### 输入Technician的工作时长，以及SalesMan和SalesManager的销售额
![alt text](<屏幕截图 2024-10-30 214957.png>)
#### 5 => 展示所有员工的薪酬（以表格形式）
##### 以ID顺序展示员工薪酬，同时列出对应等级
![alt text](<屏幕截图 2024-10-30 215007.png>)
#### 0 => 退出系统
##### 系统输出“goodbye”表示退出，同时将员工数据存入employees.txt
![alt text](<屏幕截图 2024-10-30 215048.png>)
